Cheatsheet

HTML
  1. Name the components of HTML elements properly
    The components of an HTML elements is a start tag, the content of the element and the end tag.
  2. Identify void elements and how they are denoted
    Void elements don't have content or an end tag. They are denoted as empty elements.
  3. Identify attributes
    An attribute provides additional information about the HTML element. They are specified in the start tag and they are defined by a name and a value.
  4. Write down the correct DOCTYPE declaration for HTML 5
    The correct DOCTYPE declaration is <DOCTYPE! html>
  5. Write down syntactically correct statments for the following elements:
    • Articel
      The <article> specifies independent contend.
    • Body
      The <body> contains the structure of the website.
    • line break
      The <br> element will start a new line.
    • Headings
      The tags <h1> to <h6> defienes the headers of an document.
    • Section with meta-information
      The <meta> element defiens metadata of an document.
    • Top level element
      The top level element is the <html> element and containes the <head> and the <body> element.
    • Image named
      The <image> element displayes an image. The source is defined in the src attribute.
    • Paragraph
      The <p> element will start a new line and add margin before and after the element.
    • Section
      The <section> element is for general subdivision.
    • Label appearing
      The <label> element labels something.
    • Link to other pages
      The <a> element defines a hyperlink. With the hreft attribute the URL to the Page can be specified.
    • Comment
      A comment is defined by <!-- -->
    • Head with title and meta elements
      The <head> element is a container for metadata.
      The <title> element defines the title of the document which is shown in the tab.
Css
  1. Name the components of a css rule properly
    A CSS rule-set consists of a selector and a declaration block.
  2. Use combinators properly: direct child, descendant, adjacent sibling, sibling
    Combinator selectors are to select elements based on a specific relationship between them.
  3. Specify colors by color names and in hexadecimal notation
    It is recommend to use hexadecimal notation.
  4. Declaration for color, background color
    The declaration for an color or an background color is written in an CSS file or the style property.
  5. Pseudo classes for the <a> element
    Pseudo classes are defiened to specifie a special state of an element. In this example pseudo classes could be used to change the style of the link when it has been clicked.
  6. Declaration for background image
    The background-image: url("") property will change the background to the specified image or the background can be set to an color with the background-color property.
  7. Declaration for the text alignment
    The text-align property will change the alignment of the text to the left, right, center or justify, which will give each line an equal amount of space.
  8. Declaration for the text decoration
    The text-decoration property will set the text decoration to none, overline, line-through or underline the text.
  9. Declaration for text transformation
    The text-transform property will change the text to uppercase, lowercase, or capitalize.
  10. Identify the diffrence bettween serif and sans-serif fonts
    Serif fonts have small lines at the end of the characters, sans-serif does not have these.
  11. Declarations for font families
    The font-family property will change the font family.
  12. Declaration for font style
    The font-style property changes the text to normal, italic or oblique.
  13. Declaration font sizes
    The font-size property will change the size of the font to the specified pixels.
  14. Declaration for font weights
    The font-weight property will change the text to normal or bold.
