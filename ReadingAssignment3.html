Cheatsheet

Html for lists
  1. Start tags for
    • an ordered list <ol>
    • an unordered list <ul>
    • a description list <dl>
    • a list item <li>
    • a description term <dt>
    • a description data <dd>
Css for lists
  1. Make bullet a square with list-style-type:"square";
  2. Make bullet an image named "my-bullet.png" with list-style-image:"url('my-bullet.png'')";
  3. Make bullet an uppercase roman number with list-style-type:"upper-roman"
  4. Set bullet position to be inside the list item's text flow with list-style-position:"outside";
  5. Set bullet postion to be outside th list item's text flowlist-style-position:"inside";
Html for Tables
  1. Start tags for
    • a table <table>
    • a table row <tr>
    • a table header <thead>
    • a table header cell <th>
    • a table body <tbody>
    • a table cell <td>
    • a table footer <tfoot>
    • a table footer cell <td>
Css for Tables
  1. Black dotted border of 1 px width with border:"1px dotted black";
  2. Solid border of 1px width having a red color and rounded corners with border:"1px solid red"; border-radius="1px"
